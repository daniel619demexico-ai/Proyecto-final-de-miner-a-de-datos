Fundamentos de BI & Minería de Datos

Conceptos esenciales para entender la inteligencia de negocios y el descubrimiento de conocimiento.

Sociedad y paradigmas

Sociedad de la Información

Entorno donde las TIC facilitan el acceso, difusión y manipulación masiva de datos. Enfoque en la disponibilidad de información.

Sociedad del Conocimiento

Evolución hacia la transformación de información en conocimiento útil, generando innovación, aprendizaje y desarrollo social. Prioriza el valor agregado y el capital intelectual.

Inteligencia de Negocios & OLAP

Inteligencia de Negocios (BI)

Conjunto de estrategias, procesos y tecnologías para recolectar, analizar y presentar datos del negocio, apoyando la toma de decisiones estratégicas.

OLAP (Procesamiento Analítico en Línea)

Permite consultas multidimensionales rápidas (roll-up, drill-down, slicing). Ideal para análisis de datos agregados desde Data Warehouses.

Minería de Datos & KDD

Minería de datos

Proceso de descubrir patrones, correlaciones y anomalías en grandes volúmenes de datos utilizando métodos estadísticos, machine learning y sistemas de bases de datos.

Proceso KDD (Knowledge Discovery in Databases)

Metodología global para extraer conocimiento útil a partir de datos. Etapas fundamentales:

  • 1. Selección: Identificar y obtener los datos relevantes desde fuentes múltiples.
  • 2. Preprocesamiento / Limpieza: Manejo de ruido, valores atípicos, datos faltantes.
  • 3. Transformación: Reducción de dimensionalidad, discretización, creación de atributos derivados.
  • 4. Minería de datos: Aplicación de algoritmos para encontrar patrones (clasificación, clustering, reglas).
  • 5. Interpretación / Evaluación: Validar los patrones, visualizar y presentar conocimiento accionable.

Tareas & Técnicas · Correlación

Tareas comunes: Clasificación, regresión, clustering, asociación (reglas de asociación), detección de anomalías.

Técnicas: Árboles de decisión, redes neuronales, SVM, k-means, Apriori, Random Forest. Cada tarea se correlaciona con técnicas específicas: ej. clustering → k-means; clasificación → árboles de decisión; asociación → Apriori.

MOLAP, Data Warehouse, Data Marts

MOLAP (Multidimensional OLAP): almacena datos en cubos precalculados (ej. Microsoft Analysis Services).
Data Warehouse: repositorio central integrado, histórico y no volátil para reportes y análisis.
Data Marts: subconjuntos orientados a un área específica (ventas, finanzas) derivados del DW.

Vista Minable

Representación de los datos preparados, limpios y transformados, listos para ser procesados por algoritmos de minería. Incluye atributos relevantes y formato tabular o multidimensional.

Proyecto aplicado: Data Warehouse para Retail → Vistas Minables y Dashboard Estratégico

Almacén de Datos (Data Warehouse) → Vistas Minables generadas

Construcción de un DW en estrella con hechos de ventas y dimensiones (tiempo, producto, sucursal). Se realizaron procesos ETL con Pentaho y Talend. A partir del DW se crearon vistas minables para minería:

  • Vista clientes_segmentacion (edad, ingreso, frecuencia de compra) → clustering.
  • Vista transacciones_productos (cesta de compra) → reglas de asociación.
  • Vista ventas_tendencia (serie temporal + atributos) → predicción de demanda.

Técnicas de Minería de Datos Aplicadas

En base al proyecto de análisis del Data Warehouse Retail, se implementaron las siguientes técnicas:

  • Clustering K-means → Segmentación de clientes (4 grupos identificados: Premium, Ocasionales, Económicos, Leales).
  • Reglas de Asociación (Apriori) → Análisis de cesta de compra: productos que se compran juntos (ej. pan + leche → huevos).
  • Árbol de Decisión (CART) → Predicción de abandono de clientes (churn) con 87% de precisión.
  • Regresión Lineal → Pronóstico de ventas trimestrales.
